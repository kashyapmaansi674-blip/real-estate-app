# Real Estate Micro-Listing Platform

A mini property listing platform built with React.

## How to Run

1. Install Node.js from https://nodejs.org
2. Clone this repo
3. Run: npm install
4. Run: npm start
5. Open: http://localhost:3000

## Features
- Property listing page with filters (Apartment, Villa, Plot, Commercial)
- Property detail view with amenities
- Send inquiry form (Name, Email, Phone, Message)
- Map placeholder on detail page

## Database Schema
See schema.sql for the complete database design with 4 tables:
- properties
- amenities
- property_amenities
- inquiries

## Tech Stack
- Frontend: React
- Database: MySQL (schema only)
